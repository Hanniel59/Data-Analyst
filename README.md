# Data-Analyst
# Analyse des Prix des Véhicules
 
## Description
Ce projet réalise une analyse des données sur les prix des véhicules en fonction de leur marque et de leur type. Utilisant Python et des bibliothèques de traitement de données comme pandas, seaborn, et matplotlib, ce projet vise à fournir une compréhension claire des tendances des prix dans le secteur automobile.
 
## Fonctionnalités
- Importation et traitement de données de véhicules.
- Calcul des prix moyens des véhicules par marque et type.
- Visualisation des données sous forme de carte thermique pour une interprétation facile.
 
## Prérequis
Les bibliothèques Python suivantes doivent être installées :
- pandas
- seaborn
- matplotlib
- os
